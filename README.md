# react-todolist
A simple to-do list I built to learn the fundamentals of React. User can add tasks, check/uncheck tasks, see number of remaining tasks, and clear completed tasks.

## Instructions to run and download
1. Download node if you don't have it already: https://www.youtube.com/watch?v=VShtPwEkDD0&t=0s 
2. Clone repository.
3. Navigate to the root directory of app.
3. Run `npm start` in terminal to run application.
4. Navigate to given localhost URL through browser.
